This Corgi Life

This Corgi Life is a fictitious social networking site. The purpose of this project was to install WordPress on an Ubuntu server and experiment with some of the different themes and plugins available to developers.

The PeepSo plugin was used as the foundation for the social networking functionality of the site while the theme, VW Landing Page, was used as the template for much of the styling.

Other plugins used to optimize and provide an extra layer of security include Autoptimize, Imagify, Limit Login Attempts Reloaded, and Wordfence.

WordPress. LAMP stack. Ubuntu server.
